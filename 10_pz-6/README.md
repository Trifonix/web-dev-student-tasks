# Моря России — страница с фреймами

## Описание задания
Создать веб-страницу с использованием **фреймов**, где:
- в левом фрейме располагаются ссылки на разные моря России,
- в правом фрейме отображается фотография и информация о выбранном море.

## Структура проекта
- **index.html** — основной файл с фреймсетом (делит экран на левый и правый фреймы).  
- **nav.html** — левый фрейм с перечнем морей России. Ссылки открывают страницы в правом фрейме.  
- **default.html** — стартовая страница правого фрейма с приветствием и инструкцией.  
- **sea1.html — sea5.html** — страницы с фотографиями и описанием каждого моря.  
- **style.css** — стили для навигации, правого фрейма и приветственной страницы.  

## Функциональные требования
- Левый фрейм содержит вертикальный список ссылок.  
- При клике на ссылку в правом фрейме отображается соответствующая информация и изображение.  
- Ссылки имеют визуальный эффект при наведении (подсветка и изменение цвета текста и фона).  
- Если браузер не поддерживает фреймы, выводится сообщение с предложением обновить браузер.  

## Особенности интерфейса
- Левый фрейм: список ссылок с рамкой и плавным эффектом наведения.  
- Правый фрейм: блок с изображением и текстовой информацией о выбранном море.  
- Начальная страница правого фрейма приветствует пользователя и даёт подсказку, как использовать навигацию.  

## Дополнительно
- Страницы оформлены единым стилем: скруглённые углы, тени, аккуратные рамки.  
- Изображения морей (`sea1.png`, `sea2.png` и т.д.) размещаются в корне проекта или в отдельной папке с изображениями.
