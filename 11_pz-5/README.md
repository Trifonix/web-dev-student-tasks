# Контактная форма

## Описание
Минималистичная контактная форма в чёрно-белой гамме. Форма разделена на три блока, каждый с рамкой и заголовком сверху.

### Разделы
1. **Контактная информация** — имя, телефон, email (обязательные поля).  
2. **Персональная информация** — возраст, пол (выпадающий список), личные качества.  
3. **Любимые животные** — выбор нескольких вариантов с помощью чекбоксов.  

### Особенности
- Чёрно-белый минималистичный дизайн.  
- Каждый блок визуально отделён рамкой с заголовком.  
- Поле «Пол» выполнено как выпадающий список.  
- Кнопки отправки и очистки с эффектом при наведении.  

### Структура проекта
- **index.html** — страница с формой.  
- **style.css** — стили для оформления формы.
