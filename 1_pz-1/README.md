# Практическая работа №1

## Описание задания

Цель данной работы — закрепить навыки работы с HTML-элементами и CSS-стилями для их оформления.

Вам необходимо создать HTML-страницу с заголовками, таблицами и абзацами текста и применить к ним различные CSS-стили.

### Задание 1

1. Установите выравнивание всех заголовков `<h1>` по правому краю страницы.  
2. Установите выравнивание всех заголовков `<h2>` по центру.  
3. Задайте всем таблицам фиксированную ширину в 500 пикселей.  
4. Выравнивание текста в заголовках столбцов таблиц (`<th>`) должно быть по левому краю.  
5. Выравнивание текста в ячейках таблиц (`<td>`) должно быть по центру.  
6. Ограничьте ширину всех абзацев текста до 300 пикселей.  
7. Выровняйте текст всех абзацев одновременно по левому и правому краю (justify).  
8. Установите заголовкам `<h2>` ширину 300 пикселей и центрируйте их внутри этого блока.

### Задание 2

Для второго блока страницы:

1. Заголовку `<h1>` установите размер шрифта 50px и выравнивание по центру.  
2. Заголовкам `<h2>` установите размер шрифта 40px и отмените жирность текста.  
3. Абзацам установите размер шрифта 30px.
