# Практическая работа 1. Верстка текста и заголовков

## Цель работы
Научиться создавать простую веб-страницу с заголовками, абзацами и различными стилями текста, используя HTML и CSS.

## Описание задания
Необходимо реализовать верстку страницы, которая будет содержать заголовки и текстовые блоки, оформленные с помощью CSS. На странице должны присутствовать:

1. **Главный заголовок страницы (h1)** с крупным размером шрифта.  
2. **Второстепенные заголовки (h2)** с меньшим размером шрифта и без жирного начертания.  
3. **Абзацы текста (p)** с определенным размером шрифта и межстрочным интервалом.  
4. Использование **курсива** для выделения частей текста и **жирного начертания** для ключевых слов или понятий.

## Требования к верстке
- Подключить внешний CSS-файл (`style.css`) для оформления страницы.  
- Установить шрифты, размеры и отступы для всех элементов: заголовков и абзацев.  
- Для отдельных слов или фраз использовать стили `bold` (жирный) и `italic` (курсив) через CSS-классы.  
- Обеспечить читаемость и аккуратный визуальный вид текста с помощью отступов между блоками и заголовками.

## Структура страницы
- Введение с описанием задания.  
- Основной заголовок (h1) с определением термина CMS.  
- Несколько абзацев текста с пояснениями и примерами использования CMS.  
- Второстепенные заголовки (h2) для разделения смысловых блоков:  
  - «Какие бывают CMS»  
  - «Примеры CMS»  
- Абзацы с текстом, содержащие ключевые термины, выделенные курсивом и жирным шрифтом.

## Ожидаемый результат
- Страница должна выглядеть аккуратно и структурировано.  
- Заголовки и абзацы должны соответствовать заданным стилям.  
- Использование CSS должно быть внешним (файл `style.css`).  
- Основной текст должен быть легко читаемым, ключевые элементы выделены.
