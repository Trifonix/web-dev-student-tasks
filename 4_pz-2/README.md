# Практическая работа: CSS-селекторы и оформление элементов

## Цель работы
Научиться применять CSS-селекторы для изменения стилей элементов на веб-странице, понимать работу классов и иерархических селекторов.

---

## Задания

### 1. Покрасьте содержимое div
Дан HTML-код с четырьмя блоками `<div>`.  
**Задача:**  
- Содержимое первого и третьего блока покрасить в красный цвет.  
- Содержимое второго и четвертого блока покрасить в зеленый цвет.

---

### 2. Изменение размеров заголовков
Дан HTML-код с заголовками `<h2>` и `<h3>`, все они имеют класс `header`.  
**Задача:**  
- Сделать все `<h2>` с классом `header` размером 40px.  
- Сделать все `<h3>` с этим классом размером 30px.

---

### 3. Понимание CSS-селектора
Дан CSS-селектор:  
`.bbb .zzz { color: red; }`  
**Задача:**  
- Объясните, какие элементы он выбирает.  
- Напишите HTML-код, подходящий под этот селектор.  

*Подсказка:* селектор выбирает все элементы с классом `zzz`, которые находятся внутри элемента с классом `bbb`.

---

### 4. Селектор по началу класса
**Задача:**  
- Напишите CSS-селектор, который выбирает все абзацы `<p>`, у которых класс начинается с `top-`.  
- Объясните принцип работы такого селектора.

---

## Рекомендации
- Используйте селекторы по типу, классу, иерархии и атрибутам в зависимости от задачи.  
- Проверяйте стили в браузере, чтобы убедиться, что они применяются корректно.  
- Следите за семантикой HTML и структурой документа.
