# Практическое задание 2: Организация ссылок на веб-странице

## Цель работы
Научиться создавать HTML-страницы с заголовками, ссылками и изображениями, корректно организовывая структуру папок и файлов.

## Задачи

1. Создать папку **Lab**.
2. В папке **Lab** создать три HTML-файла:
   - `index.html` — главная страница.
   - `author.html` — страница "Об авторе".
   - `image.html` — страница с картинкой.
3. Страницу `author.html` поместить в папку **author** внутри **Lab**, остальные страницы оставить в корне.
4. **index.html**:
   - Заголовок 1 уровня, выровненный по центру: *«Практическое задание 2»*.
   - Заголовок 2 уровня, выровненный по центру: *«Организация ссылок на веб странице»*.
   - Две ссылки, выровненные по центру: на `image.html` и `author.html`.
5. **author/author.html**:
   - Заголовок 1 уровня: *«Об авторе»*.
   - Информация о себе.
   - Ссылка на главную страницу (`index.html`).
6. **image.html**:
   - Картинка по центру страницы, ширина 500px.
   - Ссылка на главную страницу (`index.html`).

## Результат
- Корректная структура папок и файлов.
- Центрированные заголовки и ссылки.
- Страница с картинкой, выровненной по центру.
- Логичная навигация между страницами.
