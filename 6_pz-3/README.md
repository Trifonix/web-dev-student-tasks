# Практическая работа CSS

## Задания

### 1. Курсив в списках и абзацах
- Дан HTML код с `<ul>` и `<p>`, содержащими `<i>`.
- Требуется:
  - Курсив внутри `<ul>` покрасить в красный.
  - Курсив внутри `<p>` покрасить в зеленый.

### 2. Жирный текст внутри li
- Дан HTML код с `<li>`, содержащими `<b>` и `<i>`.
- Требуется покрасить в красный только те `<b>`, которые являются непосредственными потомками `<li>`.

### 3. Селектор `.eee p.bbb`
- Объясните, что выбирает селектор `.eee p.bbb`.
- Напишите HTML код, который подходит под этот селектор.
- Стили: текст красного цвета.

### 4. Следующий элемент после `.elem`
- Дан HTML код с классами `.elem` на некоторых `<li>`.
- Напишите селектор, который выберет элементы, находящиеся **сразу после** элементов с классом `.elem`.

### 5. Ссылки
- Сделайте все ссылки `<a>`:
  - В обычном состоянии голубыми и без подчеркивания.
  - В состоянии `:hover` — красными и подчеркнутыми.

### 6. Наведение на абзацы
- Все `<p>` при наведении мыши должны менять цвет текста на синий.

### 7. Кнопка
- Создайте кнопку `<button>`.
- При нажатии на кнопку (`:active`) фон должен становиться зелёным (`background-color`).

### 8. Список с эффектами hover
- Создайте список из 10 элементов `<li>`.
- Установите синюю рамку вокруг каждого элемента и небольшой отступ между ними.
- При наведении мыши:
  - Выделенный элемент окрашивается в зеленый.
  - Все элементы ниже выделенного — в красный.

### 9. Первое предложение красное и курсивное
- Создайте 4 абзаца текста.
- Первое предложение каждого абзаца должно быть красным и курсивным.

### 10. Текст в начале и в конце абзаца
- В начале каждого абзаца средствами CSS вставьте текст `"Начало:"` красного цвета.
- В конце каждого абзаца вставьте текст `":конец!"` зелёного цвета.
- Создайте 4 абзаца с таким оформлением.

### 11. Каждая третья li красная
- Создайте список `<ul>` или `<ol>`.
- Каждая третья `<li>` должна быть красного цвета.
