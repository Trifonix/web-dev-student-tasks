# Практическая работа: CSS фигуры

## Задания

### 1. Вложенные прямоугольники
- Создать три прямоугольника, каждый внутри другого:
  - Внешний: красная рамка (сплошная)
  - Средний: зелёная рамка (пунктир)
  - Внутренний: синяя рамка (пунктир)
- Использовать `div` и классы для стилизации.

### 2. Абзацы с рамками
- Создать два абзаца с текстом `Lorem ipsum`:
  - Первый: зелёная рамка на всю ширину страницы
  - Второй: красная рамка, ширина 30%, по центру страницы

### 3. Красный круг
- Создать красный круг с пустым внутренним пространством.
- Использовать `border-radius: 50%` и красную рамку.
