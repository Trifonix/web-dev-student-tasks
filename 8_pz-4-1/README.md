# Практическая работа №4-1: Таблицы

## Описание работы
В рамках работы были созданы три веб-страницы (объединенные в один `index.html`) с таблицами:

1. **Тригонометрическая таблица**  
   - Содержит углы α и значения sin, cos, tg, ctg.  
   - Выделена строка с 45°.

2. **Таблица продаж канцелярских товаров**  
   - Таблица с группировкой товаров, датами и магазинами.  
   - Каждая группа товаров имеет свой цвет фона.  
   - Таблица адаптивная по ширине с тенями и эффектом hover.

3. **Таблица студентов**  
   - Узкая таблица (30% ширины окна).  
   - Содержит имя, фамилию и курс.  
   - Есть эффекты подсветки строк при наведении.

## Используемые технологии
- HTML5 для структуры страниц.  
- CSS3 для стилизации таблиц, hover-эффектов и цветовых выделений.
