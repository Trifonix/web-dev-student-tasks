# Практическое задание №5

## Описание работы

В работе реализованы три задания:

1. **Горизонтальное меню (эффект "Новости")**  
   - Меню из 5 элементов: Главная, Новости, Контакты, О компании, Как добраться.  
   - Синий текст, рамка чуть темнее синего.  
   - При наведении: тёмно-серый фон, бледно-белый текст.  
   - Отступ между элементами 2px.  

2. **Второе меню (эффект "PHOTOS")**  
   - Меню из 6 элементов: HOME, BLOG, PHOTOS, ABOUT, LINKS, CONTACT.  
   - Общая оранжевая полоса меню.  
   - При наведении: красный фон кнопки, текст бледно-жёлтый.  

3. **Блочные ссылки**  
   - Несколько ссылок ведут себя как блочные элементы.  
   - Заданы граница, ширина, высота и эффект наведения.
